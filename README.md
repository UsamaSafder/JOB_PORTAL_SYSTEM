# JOBPORTALSYSTEM

This repository contains a React frontend in `src/react` and a Node/Express backend in `Backend`.

## Development server

To start the React frontend from the repository root, run:

```bash
npm install
npm start
```

The frontend runs with Vite on `http://localhost:5173/` by default.

## Start frontend + backend together

To run both the React frontend and the backend from the repository root:

```powershell
npm install
npm run start:all
```

This starts the frontend with Vite and the backend API in `Backend/` concurrently.

## Building

To build the React frontend, run:

```bash
npm run build
```

The production build output is generated by Vite inside `src/react/dist`.

## Backend

To run only the backend API:

```bash
npm run backend
```

## Testing
<br>
The root `npm test` script currently performs a frontend build verification.
